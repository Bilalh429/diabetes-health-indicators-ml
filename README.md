## Project Overview
This project uses the Diabetes Health Indicators Dataset to build and evaluate predictive machine learning models for healthcare-related outcomes. The dataset includes clinical, lifestyle, and diagnostic data from patients.

### Key Tasks
- **Binary Classification**: Predict diabetes diagnosis (Yes/No)
- **Multiclass Classification**: Predict diabetes stage (No Diabetes, Pre-diabetes, Diabetes)
- **Regression**: Predict continuous outcomes (risk score, glucose level)
